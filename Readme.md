# 🚀 Blogify - Modern Blog Platform (PWA Ready)

Blogify is a **responsive blog web application** built using **React, Tailwind CSS, and Vite**.  
It allows users to create, view, and manage blogs with a clean UI and supports **PWA (installable app)** functionality.

---

## ✨ Features (Current Project)

### 📝 Blog System
- Create blog posts
- Save blogs using localStorage
- Display fake + user-created blogs together

### 📚 Blog Listing
- Responsive blog grid layout
- Blog cards with title & description
- "Read More" button UI

### 🔍 Search Functionality
- Live search blogs by title
- Instant filtering results

### 📊 Dashboard
- Total blogs count
- Fake analytics (views & comments)
- Modern gradient UI cards

### 🎨 UI / UX
- Fully responsive design (mobile, tablet, desktop)
- Dark modern theme
- Smooth hover animations
- Framer Motion hero animation

### 📱 PWA Support
- Installable web app
- Works like a mobile application
- Service worker enabled
- Offline caching support (basic)

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- vite-plugin-pwa

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/your-username/blogify.git
````

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

---

## 📦 Build Project

```bash
npm run build
```

---

## 📱 PWA (Install App)

This project supports PWA.

### How to install:

1. Open in Chrome
2. Click "Install App"
3. Use Blogify like a mobile app 📱
